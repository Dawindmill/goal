Our System Design

Take a look at some of our recent products

In our approach to architecture of the system, there are two main considerations - infrastructure design and application software component design and integration. The infrastructure setup gives us facility to house our application requirements, tools and frameworks. The key aspect for infrastructure is to be able to cope with growing user base and, scale up with more demanding analytic cases. For application design, we decompose engineering tasks into software components and integration of these software modules to serve the different use cases. Another key aspect of the system design is to be able to utilize Cloud Computing and build on these Cloud Platforms in mind. With these design consideration, we can possible achieve highly available, reliable and scalable distributed system.
