GOAL project description

This is a project for Clusters and Cloud Computing subject in University of Melbourne.


We built harveter to grab tweets and store them in CouchDB, after that we use distribtued MapReduce approch to process the data to perform sentiment anlaysis for the topics, crime detection and the positive rate of people supporting Trump.


There is an Automator made to automatically boot and setup instances with CouchDB as our base infrastructures with Boto and Ansible. The web application, sentiment analyzer program, twitter harvesters are also deployed in an automatic way.
